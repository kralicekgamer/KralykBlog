Obecné pojmy

  • WWW (World Wide Web) - systém pro prohlížení dokumentů nacházejících se na internetu
  • WiFi (Wireless Fidelity) - bezdrátové připojení k počítačovým sítím
  • ISP (Internet Service Provider) - poskytovatel připojení k internetu

Reprezentace dat

  • bit - nejednoduší reprezentace informací, (0, 1)
  • nibble - ozačuje 4 bity
  • bajt - označuje 8 bitů
  • kilobyte (KB) - 1 024 bajtů
  • megabyte(MB) - 1 024 KB
  • gigabyte (GB) - 1 024 MB
  • terabyte (TB) - 1 024 GB
  • petabyte (PB) - 1 024 TB

pozn.: Pokud jsou jednotky uvedeny malými písmeny, například kb, Kb nebo kB, jedná se o kilobity (kb), což odpovídá 1000 bitům. Tyto jednotky se často používají k vyjádření rychlosti přenosu dat, například v Kbps(kilobity za sekundu).

Protokoly

  • Ethernet - lokální síť
  • TCP/IP (Transmission Control Protocol/Internet Protocol) - internetové sítě
    • IPv4 (Internet Protocol version 4) - protokol pro adresování sítě
    • IPv6 (Internet Protocol version 6) - novejší verze IPv4
  • HTTP (HyperText Transfer Protocol) - hypertextové dokumenty (weby)
  • TLS/SSL (Transport Layer Security/Secure Sockets Layer) - protokoly pro šifrování komunikace mezi webovým serverem a klientem.
  • FTP (File Transfer Protocol) - soubory
  • SMTP (Simple Mail Transfer Protocol) - e-maily
  • POP3 (Post Office Protocol 3) - přijímání e-mailů
  • IMAP (nternet Message Access Protocol) - příjimání a správa e-mailů
  • DNS (Domain Name System) - překlad doménových jmen na IP
  • SSH (Secure Shell) - přenos dat mezi PC

Zařízní

  • switch - propojení zařízení v síti
  • router - směruje data mezi různými sítěmi
  • server - poskytování služeb pro ostatní uživatele

Networking

  • oketa - část IP oddělená tečkami

  • maska podsítě - určuje část IP která patří síti

  • CIDR Classless Inter-Domain Routing) - formát zápisu IP a podsítě

  • DHCP (Dynamic Host Configuration Protocol - automatické přidělování IP adres zařízením

  • localhost - odkaz na právě používaný PC

  • gateway - spojení dvou různých sítí

  • default gateway - brána pro směrování mimo lokální síť (router)

  • síťový uzel - přijímá, zpracovává nebo přeposílá data v síti

  • backbone - hlavní síťová infrastruktura, která propojuje různé části sítě

  • PAN (Personal Area Network) - osobní zařízení v blízkosti

  • LAN (Local Area Network) - v domovech, firmách

    • WLAN (Wifi Local Area Network) - bezdrátová síť LAN
    • VLAN (Virtual Local Area Network) - virtuální síť, která rozděluje LAN na menší části.

  • MAN (Metropolian Area Network) - městská oblast

  • WAN (Wide Area Network) - propojuje města, státy

  • GAN (Global Area Network) - propojuje WAN

Základní síťové služby

  • NAT (Network Address Translation) - technika pro překládání adres mezi vnitřní a vnější IP adresou.
  • Load Balancer - zařízení nebo software pro rozdělení zátěže mezi více serverů.
  • Proxy server - server, který zprostředkovává komunikaci mezi klientem a cílovým serverem.

Typy připojení k internetu

  • Fiber Optic - připojení k internetu pomocí optických vláken pro velmi vysoké rychlosti.
  • DOCSIS - připojení přes kabelovou televizi.
  • HFC (Hybrid Fiber-Coaxial) - kombinace optiky a koaxiálních kabelů.
  • WiMAX- bezdrátové připojení na dlouhé vzdálenosti.
  • Satelitní internet - připojení přes satelit, ideální pro venkovské oblasti.
  • DSL - připojení přes telefonní linku.
